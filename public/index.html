Go to Arcus

This is my project: Arcus! It's a website intended to categorize and display each and every one of the Discord chat bots on the planet. Arcus is still in its early stages of course, but it's at a fantastic baseline for further refinement.

For the uninformed, Discord is a voice and chat client that's been taking the gamer community by storm because it keeps doing right what every other major chat/voice client has been doing wrong for the past decade. Along with its uprisal, countless developers have designed their own chat bots that Discord users can add to their servers. These bots have all kinds of features from telling you the weather in a second, to playing music in your voice channel.

Arcus allows developers to advertise their bots among its indices. They control the description, information, etc, and Arcus handles the rest, providing an easy "Add to Server" link, and grabbing profile images and data off the discord API.

Arcus allows everyday discord users to browse all the bots it has to offer through the "Go to Bots" page. This includes a search utility that can help them meet their needs quicker. They can also expand the view for a particular bot to see what others have commented on it. Likewise, they can view a developers profile to see their bots, and what people think about them in comments. If a user registers/logs in, either through the classic username/password, or via Discord API authentication, they too can comment on bots and developers, as well as manage their public profile.

A few notes:

  • If you see an empty circle where a profile image should be, that means the user or the bot doesn't yet have one configured.
  • The Add to Server button requires some specific information from the bot application on discord, and if some of the values are provided, the link won't work. This is on the developers to ensure.
